# The Oscar Movies App

The Oscar Movies is a web application that allows users to search for movies and view popular movie details using The Movie Database (TMDb) API. The app displays movie information such as the title, poster, and vote average.

## Features

- **Search Movies**: Users can search for movies using the search bar.
- **Popular Movies**: Displays a list of popular movies by default.
- **Movie Details**: Shows the title, poster, and vote average of each movie.

## Technologies Used

- **HTML**: Markup language used for structuring the web pages.
- **CSS**: Stylesheets used for the visual styling and layout of the website.
- **JavaScript**: Programming language used for the interactive elements and dynamic functionality of the website.
- **TMDb API**: Used to fetch movie data.

## Getting Started

To run the MyMovies app locally, follow these steps:

1. **Clone the repository**:

```bash

```

2. **Navigate to the project directory**:

```bash
cd mymovies-app
```

3. **Open the `index.html` file in your preferred web browser**:

```bash

```

## Usage

1. Open the app in your web browser.
2. The homepage will display a list of popular movies.
3. Use the search bar to search for specific movies by title.
4. Click the search button to view the search results.

## API Information

The app uses The Movie Database (TMDb) API to fetch movie data. You can get your own API key by creating an account on [TMDb](https://www.themoviedb.org/).

### API Endpoints

- **Popular Movies**: 
  ```
  https://api.themoviedb.org/3/discover/movie?sort_by=popularity.desc&api_key=YOUR_API_KEY&page=1
  ```
- **Search Movies**: 
  ```
  https://api.themoviedb.org/3/search/movie?&api_key=YOUR_API_KEY&query=SEARCH_QUERY
  ```
